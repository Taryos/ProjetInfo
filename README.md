# ProjetInfo

Ce dépôt regroupe plusieurs implémentations de modèles mathématiques et biologiques à l'aide de Python et du framework Taichi

## 1. Automates Cellulaires (`CellularAutomata/`)

- **GoL.py** : Jeu de la vie de Conway.
- **Lenia.py / LeniaRGB.py** : Lenia et une version avec 3 espèces chimiques
- **SmoothLife.py** : Version continue du Jeu de la vie.

## 2. Modèle de Gray-Scott (`Gray-Scott/`)
Simulation de réaction-diffusion permettant de modéliser la formation de motifs biologiques et son problème inverse
- **train.py** : Script d'entraînement
- **Motifs.py** : Simulation de Motifs
